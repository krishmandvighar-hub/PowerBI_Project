# 📊 Shakti Ecommerce Sales Dashboard | Power BI

An interactive Power BI dashboard developed to analyze ecommerce sales performance using real-world business data. This dashboard provides valuable insights into sales, profit, customer behavior, payment methods, product categories, and monthly trends, helping businesses make informed decisions through data visualization.

---

## 📌 Project Overview

The Shakti Ecommerce Sales Dashboard transforms raw sales data into meaningful insights using Power BI. It enables users to monitor key performance indicators (KPIs), identify sales trends, analyze customer purchasing behavior, and evaluate business performance with interactive reports.

---

## ✨ Dashboard Features

- 💰 Total Sales Amount
- 📦 Total Quantity Sold
- 📈 Total Profit
- 💳 Average Order Value (AOV)
- 🗓️ Quarterly Filter (Q1–Q4)
- 🌍 Sales by State
- 💳 Payment Mode Distribution
- 👕 Category-wise Quantity Analysis
- 📈 Monthly Profit Trend
- 👥 Top Customers by Sales
- 📦 Profit by Product Sub-Category
- 🎛️ Interactive Filters and Slicers

---

---

## 📂 Dataset

The dashboard is built using two datasets:

- Orders.csv
- Details.csv

These datasets were cleaned, transformed, and modeled in Power BI before creating the final dashboard.

---

## 📈 Key Insights

- Maharashtra generated the highest sales among all states.
- Cash on Delivery (COD) is the most preferred payment method.
- Clothing contributes the highest share of product quantity sold.
- Printers generated the highest profit among all product sub-categories.
- Monthly profit analysis helps identify high-performing and low-performing months.
- Customer-wise sales analysis highlights the top revenue-generating customers.
