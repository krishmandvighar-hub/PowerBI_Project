# PowerBI_Project

📊 Shakti Ecommerce Sales Dashboard | Power BI

An interactive Power BI dashboard built to analyze Shakti Ecommerce sales performance using real-world business metrics. This dashboard provides insights into sales, profit, customer behavior, payment methods, product categories, and monthly performance, helping businesses make data-driven decisions.

🚀 Project Overview

This dashboard transforms raw ecommerce data into meaningful visualizations, enabling users to monitor KPIs and identify business trends quickly.

📂 Dataset

The project uses two datasets:

Orders.csv
Details.csv

These datasets were cleaned, transformed, and modeled in Power BI before creating the dashboard.

📌 Key Insights
Maharashtra generated the highest sales.
COD is the most preferred payment method.
Clothing contributes the largest share of total quantity sold.
Printers generated the highest profit among all sub-categories.
Monthly profit trends help identify peak and low-performing periods.
